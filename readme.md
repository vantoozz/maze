# Maze Generator

```shell
➜ maze
Usage: app [OPTIONS]

Options:
  -w, --width INT            Maze width
  -h, --height INT           Maze height
  -f, --format [ASCII|JSON]  Output format
  --help                     Show this message and exit

```
